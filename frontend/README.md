# Frontend Documentation

## Installation

1. **Clone the repository:**
   ```sh
   git clone <repository-url>
   cd <repository-directory>
   ```
2. **Install the dependencies:**
   ```sh
    npm install
    ```
3. **Set up the env**
- Create `.env` as instruction in `.env.example`
   
## Usage
**Start the server:**
   ```sh
   cd <repository-directory>/frontend>
   npm run dev
   ```
